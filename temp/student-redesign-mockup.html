What does "abundant" mean?

✗ Yours Moving quickly from place to place
✓ Correct Existing in large amounts; plentiful
💡 Why?
"Abundant" means there is a lot of something. Think of an abundant harvest — so many fruits and vegetables that there's more than enough for everyone!
📚 Example
The garden had an abundant supply of tomatoes this summer.

Summary of All Changes

#1 DM Sans font Applied to all student portal text. Rounder, friendlier letterforms vs system font stack.
#2 Prominent review card Subtle gradient background + stronger shadow draws the eye to the primary action.
#3 Generous spacing Larger card padding (28px), bigger option gaps (12px), 2px option borders, more margin between feedback sections.
#4 Fade-in transitions Cards fade in with a subtle upward slide (0.3s) when questions change. Reduces visual "flash."
#5 Standardized radii Only 3 values: 14px (cards), 12px (buttons), 999px (pills). Replaces the current mix of 10/12/14/16/18px.
