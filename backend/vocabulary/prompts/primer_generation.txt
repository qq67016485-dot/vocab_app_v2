- term, part_of_speech, definition (formal), example_sentence

Your job is to create a kid-friendly version of the definition and a syllable breakdown.

Output valid JSON with this exact structure:
{{
  "primer_cards": [
    {{
      "term": "the word",
      "syllable_text": "syl·la·ble form",
      "kid_friendly_definition": "a simple definition a child can understand"
    }}
  ]
}}

Guidelines:
- Base the kid-friendly definition on the provided formal definition
- Kid-friendly definitions should use simple vocabulary (under 600L Lexile)
- Syllable text uses the middle dot character (·) as separator
- Keep definitions concise (1-2 sentences max)
- Do not generate a new example sentence — the formal example will be reused
