# Script Learning Behavior Runtime Guide Plan

Status: future runtime-canon guide
Purpose: define a compact guide that can load alongside selected visual sheets during graphic novel script generation

## Goal

Create a future `script-learning-behavior.md` guide that helps the script model turn vocabulary knowledge into story action without making characters deliver textbook definitions.

The guide should apply to all Lexi Legends stories, with character-specific examples where useful. It should remain separate from visual sheets so image generation payloads stay focused on visual continuity.

## Content To Include

- Vocabulary meaning changes the story outcome: opens a route, repairs a misunderstanding, reveals a hidden pattern, changes a character's interpretation, or solves a concrete realm problem.
- Characters test words through evidence: context clues, roots, prefixes, suffixes, near-synonym contrast, tone, sentence fit, and visible consequences.
- Partial theories are useful when marked as tentative, tested aloud, and revised.
- Folio can nudge with a short question or visual note, but should not lecture.
- Target words should be used naturally in dialogue, narration, or in-world text.
- Avoid dictionary-style exposition and lines like "I learned that this word means..."

## Character-Specific Hooks

- Amara models evidence-based word reasoning: circles clues, compares near-synonyms, and revises theories.
- Leo models creative trial and correction: first tries may be messy, then become precise.
- Mei models speed with accuracy: quick choices only work when the meaning fits.
- Hugo models careful checking: slow rereading and context testing can be powerful.
- Folio models supportive prompts: brief hints, definitions only when natural, and review-page curation.

## Runtime Use

Load this future guide during script generation only. Do not load it into image generation unless the image prompt needs a specific visual behavior from the script.

Keep the guide short enough to combine with selected character visual sheets and relevant setting sheets.
